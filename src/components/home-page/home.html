ts-wc-template

A modern starting point for building TypeScript Web Components with Vite.

View To-Do List Demo

Key Features

  • Vite for Fast Development
  • TypeScript Ready
  • Web Component Based
  • Simple Routing
  • Tailwind CSS Integrated
  • Unit Testing with Vitest

Get Started

Clone the repository and start building your next web application:

git clone https://github.com/sholtomaud/ts-wc-template.git
cd ts-wc-template
npm install
npm run dev

Explore the documentation on GitHub.
